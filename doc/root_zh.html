Go is …

… simple 简洁

package main

import "fmt"

func main() {
  fmt.Printf("Hello, 世界\n")
}

… fast 快速

Go compilers produce fast code fast. Typical builds take a fraction of a second yet the resulting programs run nearly as quickly as comparable C or C++ code.

Go编译器可以快速编译代码。增量编译时，通常可以在1秒内编译完成。并且得到的go程序运行速度 和C/C＋＋相近。

… safe 安全

Go is type safe and memory safe. Go has pointers but no pointer arithmetic. For random access, use slices, which know their limits.

Go是类型安全的，也是内存安全的。Go语言中提供指针，但是没有指针运算。如果想随机访问 某段内存中的数据，可以用slices（它知道内存的有效区域）。

… concurrent 并行编程

Go promotes writing systems and servers as sets of lightweight communicating processes, called goroutines, with strong support from the language. Run thousands of goroutines if you want—and say good-bye to stack overflows.

作为一种系统级语言，Go语言提供了一种叫goroutines的可通讯的轻量级线程的概念。你可以启动上千个 goroutines，并且永远不用再担心堆栈溢出的问题。

… fun 功能强大

Go has fast builds, clean syntax, garbage collection, methods for any type, and run-time reflection. It feels like a dynamic language but has the speed and safety of a static language. It's a joy to use.

Go程序编译快，语法清晰，完善的垃圾内存收集，每种类型可以定义自己的方法，运行时提供反射支持。 它有很多动态语言的特性，但是速度上和静态编译语言类似。用Go编程是一种享受 :)

… open source 开源
